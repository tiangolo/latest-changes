# Latest Changes

Automatically add the changes from a PR to the release notes in a file.

## How to use

Install this GitHub action by creating a file in your repo at `.github/workflows/latest-changes.yml`.

A minimal example could be:

```yml
name: Latest Changes

on:

...
```
